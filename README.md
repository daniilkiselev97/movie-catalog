- Клонировать репозиторий:
  git clone https://github.com/daniilkiselev97/movie-catalog
  cd movie-catalog

- Установить зависимости:
  npm install

- Запуск сервера (json-server):
  npn run start:api

- Запуск клиентской части (Angular):
  npm start

- Папка answers:
  В директории 'answers' находятся ответы на теоретические вопросы
